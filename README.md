# Gerald's Private Web

Website pribadi Gerald Jonathan William.

Dibangun pake HTML, CSS, dan JavaScript murni — no framework, no bullshit.

## Stack

- HTML5
- CSS3 (iOS 26-inspired Liquid Glass aesthetic)
- Vanilla JavaScript (ES5 compatible)
- Font: Inter (Google Fonts)

## Struktur File

```
/
├── index.html          → Halaman utama
├── style.css           → Semua styling
├── script.js           → Semua logic & interaksi
├── config.json         → Konfigurasi (coming soon)
├── README.md           → Lo lagi baca ini
└── assets/
    └── deskripsi/
        ├── desk-singkat.md     → Deskripsi singkat tentang Gerald
        └── instruction.md     → Instruksi pengisian konten
```

## Design System

| Token        | Value                     |
|--------------|---------------------------|
| Base BG      | `#0a0a0f`                 |
| Text Primary | `#ffffff`                 |
| Accent Blue  | `#1a6fff`                 |
| Glass BG     | `rgba(255,255,255,0.05)`  |
| Glass Border | `rgba(255,255,255,0.08)`  |
| Font         | Inter (100–900)           |
| Radius       | 20px – 32px               |
| Blur         | `backdrop-filter: blur(24px)` |
